# Continuous Prisonner Dillema

The Continuous Prisonner Dillema (CPD for short) is a version of the famous Prisonner Dillema which allows varrying levels of cooperation and defiance. Instead of having 2 actions, this algorithm uses math to simulate the diversity of actions.

In this base algorithm is included:
1. A simple tit for tat algorithm, which simply copies the opponent
2. A "trump" algorithm which uses anger levels in their actions
3. A smart algorithm to beat said trump algorithm

Everything is packaged up in a nice svelte server, allowing you to analyse the data using graphs on the go.
